# multi_tool_agent 功能测试文档

## 1. 测试什么
- 功能名称：多工具 Agent（`src/phase2_multi_tool_agent.py`）
- 测试点编号：MTA-01
- 测试目标：验证 Agent 可根据问题自动调用工具并返回结果

## 2. 怎么测试
### 2.1 运行环境
- OS：Windows 10/11
- Python 版本：3.10+
- 依赖版本：`pip install -r requirements.txt`
- 其他环境变量/配置：
  - API Key 已配置
  - 工具模块路径可正常导入（`src/tools/`）

### 2.2 执行脚本/命令
1. 安装依赖：
   `pip install -r requirements.txt`
2. 执行脚本：
   `python src/phase2_multi_tool_agent.py`
3. 在交互中输入测试问题：
   - 计算类：`(23 + 19) * 2 等于多少`
   - 时间类：`现在北京时间几点`
   - 天气类：`北京今天的天气如何`
4. 观察输出是否包含对应工具执行后的结果

### 2.3 前置条件/测试数据
- 前置条件：工具函数已实现且可被 Agent 注册
- 输入数据：
  - 数学表达式
  - 时间查询问题
  - 天气查询问题
- 边界数据：
  - 无法解析的问题
  - 不支持的城市天气查询

## 3. 期望结果
- 功能结果：Agent 能识别意图并调用对应工具
- 关键日志/输出：
  - 计算问题返回数值结果
  - 时间问题返回当前时间
  - 天气问题返回天气描述或合理降级提示
- 异常场景预期：
  - 非法输入时返回可理解的错误提示
  - 工具调用失败时不崩溃，给出失败原因
