# basic_call 功能测试文档

## 1. 测试什么
- 功能名称：基础模型调用（`src/basic_call.py`）
- 测试点编号：BC-01
- 测试目标：验证脚本可正常调用模型接口并返回非空响应

## 2. 怎么测试
### 2.1 运行环境
- OS：Windows 10/11
- Python 版本：3.10+
- 依赖版本：`pip install -r requirements.txt`
- 其他环境变量/配置：
  - 已配置可用 API Key
  - 网络可访问模型服务

### 2.2 执行脚本/命令
1. 安装依赖：
   `pip install -r requirements.txt`
2. 执行脚本：
   `python src/basic_call.py`
3. 观察终端输出：
   - 是否出现模型响应内容
   - 是否出现报错栈

### 2.3 前置条件/测试数据
- 前置条件：API Key 正确可用
- 输入数据：脚本内默认提示词
- 边界数据：
  - 无网络
  - API Key 无效

## 3. 期望结果
- 功能结果：命令执行成功，进程退出码为 0
- 关键日志/输出：终端打印模型返回文本，且不为空
- 异常场景预期：
  - 无网络时提示连接失败/超时
  - API Key 无效时提示鉴权失败
